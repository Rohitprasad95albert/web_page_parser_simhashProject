# Web_Pge_Parser_SEIR_Project1
Python project that fetches web pages from given url, extracts textual features like page title, page body and all links available on the page, computes 64-bit Simhash fingerprints, and compares document similarity.

# Web Page Parser and Simhash Similarity Analyzer

## Features
- Fetch web pages from command-line URLs
- Extract title, body, and links
- Compute word frequencies
- Generate 64-bit Simhash fingerprint
- Compare similarity between two web pages

## Usage
python main.py <url1> <url2>

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Regular Expressions
- Bitwise Operations
